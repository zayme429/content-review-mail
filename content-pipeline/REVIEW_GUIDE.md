# 内容审核指南

## 📋 审核流程

### 每日工作流程

```
每天 8:00
    │
    ▼
系统自动生成文章
    │
    ▼
发送通知（飞书/企微）
    │
    ▼
你收到通知后审核（30分钟内）
    │
    ├── 满意 → 回复 "发布" → 自动发布到草稿箱
    ├── 不满意 → 回复 "重新生成 [新方向]" → 重新生成
    └── 跳过 → 回复 "跳过" → 今日不发布
    │
    ▼
超时未回复 → 自动发布到草稿箱
```

## 🎯 审核指令

### 1. 发布文章
```
发布
```
或
```
确认发布
```

系统将文章发布到微信公众号草稿箱。

### 2. 重新生成（指定新方向）
```
重新生成 重点分析AI对设计师的影响
```
```
重新生成 聚焦AI工具实操教程
```
```
重新生成 从教育行业转型角度切入
```

系统会根据你的方向重新采集热点并生成文章。

### 3. 跳过今日
```
跳过
```

今日不生成文章，等待明日。

### 4. 查看文章
```
查看文章
```

系统发送文章完整内容给你。

## 📝 内容方向指引

### 六大支柱主题（循环使用）

1. **AI时代的学习方法论**
   - 如何高效学习AI知识
   - 推荐方向: "从0到1学AI的路线图"

2. **认知升级与思维模型**
   - AI时代的思维方式转变
   - 推荐方向: "系统思维在AI时代的价值"

3. **AI工具提效实战**
   - 具体工具使用技巧和案例
   - 推荐方向: "Cursor/Claude实战案例"

4. **职业转型与机会挖掘**
   - 新职业方向和发展路径
   - 推荐方向: "AI产品经理转型指南"

5. **AI时代的创造力培养**
   - 人 vs AI 的创造力差异
   - 推荐方向: "如何用AI增强创造力"

6. **信息筛选与知识管理**
   - 在海量信息中保持清醒
   - 推荐方向: "构建个人AI知识库"

## ⚠️ 质量标准

### ✅ 好文章的特征
- 有具体案例（公司名、人名、数据）
- 有独特视角（不是人云亦云的跟风）
- 有可执行建议（读者能马上行动）
- 有深度分析（不是简单罗列）
- 结构清晰（引言→现象→分析→建议→结语）

### ❌ 需要重新生成的情况
- 内容过于笼统，没有具体案例
- 观点陈旧，缺乏新意
- 语气过于营销或贩卖焦虑
- 结构混乱，逻辑不清
- 有明显的AI生成痕迹（重复、空洞）
